# ProNight Theme
Dark blue coding theme with a black terminal, modern suggest popups, and brighter icons for clear focus during late-night sessions.

## Features
- Deep navy UI chrome with bright blue accents
- Black integrated terminal with tuned ANSI palette
- Modern suggest widget and notifications
- Bolder icons and highlighted active surfaces (tabs, title bar, status bar)

## Palette Notes
- Base backgrounds: #050912 editor, #0a1320 surfaces
- Accents: #4fa3ff primary, #6fb4ff hover
- Text: #dfe6f5 primary, #e7eefc icons
- Warnings/Errors: #f2c078 warning, #ff7b72 error
